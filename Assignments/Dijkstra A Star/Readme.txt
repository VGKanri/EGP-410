Compile/Linking the project:
Make sure DeanLib is updated for visual studio 2015
Build each project in the solution
Either right click and run debug on editor or game OR set one as the default startup project

Keyboard Input:
A - switch pathfinding algorithm to A*
D - switch pathfinding algorithm to Dijkstra
Left Mouse - set the start node
Right Mouse - set the goal node
Esc - Exit program

Additional Maps can be constructed using the editor and saved using the S key
